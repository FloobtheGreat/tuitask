# tuitask

`tuitask` is a local-first terminal task manager built with TypeScript, React, Ink, and SQLite. It is designed for fast keyboard-driven task management, with priorities, due dates, descriptions, completion tracking, filtering, and responsive task details. Data is stored locally in the platform-standard user data directory, with `TUITASK_DB_PATH` available as an override.

The project is under active development. Run `npm run dev` to launch it locally or `npm run verify` to run typechecking, linting, formatting checks, tests, and the production build.
